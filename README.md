# Examples of ECharts

## Install

```shell
npm install
```

## Dev

```shell
npm run dev
```

## Release

```shell
npm run release
```

It will copy all the build resources to incubator-echarts-website/next/examples

## Update example snapshots

```shell
node tool/build-example.js
```

Only for default theme

```shell
node tool/build-example.js -t default
```
